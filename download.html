Binary/Source/Helpは，SourceForgeの リリースシステムを使用しています．sakura.exeなど個別ファイルのダウンロードは SoruceForge Project Page からどうぞ．

サクラエディタはあとで外部ツールを追加して機能拡張することができます．それら関連ファイルを含めた導入方法については こちらをご覧ください． 関連ファイルをまとめてダウンロード＆インストールする Sakura Down もあります．

※利便性・安定性の観点から、ファイルの運用場所を SourceForge から GitHub に移転準備中です。移転が完了し次第、本ダウンロードページにおけるリンクも差し替えていきます。

パッケージダウンロード

本サイトの更新は遅れることがあります．パッケージの更新状況は SoruceForge Project Page でご確認ください．

拡張ツール

エディタの機能のうちのいくつかは外部ツールを必要とします。バイナ リとは別に以下のツールも併せてダウンロードしてください。

bregonig.dll
正規表現検索・置換および正規表現キーワード強調表示に必要で す．(インストーラパッケージには含まれています)
V1(ANSI版)では BREGEXP.DLLも使用可能です．
GNU diff 中のdiff.exe
Diff機能に必要です
Exuberant Ctags 日本語対応版 中のctags.exe
ダイレクトタグジャンプのtagsファイル生成に必要です．
PPA.DLL
PPAマクロの実行に必要です．
C/Migemo のmigemo.dll 及びdict/ディレクトリ以下の辞書ファイル
Migemoを使ったインクリメンタルサーチで必要です．置き場所は共通設定から変更することも出来ます．

いずれもsakura.exeと同じディレクトリに置いてください．(migemoの場合はdictディレクトリも)

実行ファイルのみダウンロード

コンパイル済み実行ファイルは SourceForge Project Folderのリリースシステムに登録されています． 更新履歴も 併せてどうぞ．

インストーラにてインストールされたものをこのファイルで上書きして も何ら問題ありません．

ソースファイルのダウンロード

リリース版のソースコード一式は SourceForgeに て公開されます。開発中のファイルは リポジトリより直接参照が可能です．

注意 ソースコードのサクラエディタ以外での取り扱いについては各Copyright Holderに個別に問い合わせてもらうことになります。

なお、zlib/libpngライセンスに従った配布が可能と書かれ ているファイルについてはその通りですので、ライセンスの範囲内で自由に転用していただいてかまいません。

ヘルプファイルのみダウンロード

ヘルプファイルの単体配布は SourceForgeにて行っています。
